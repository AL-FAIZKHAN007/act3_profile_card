# Profile Card
## Overview
This project is a simple Profile Card built using only HTML, CSS, and image. It displays basic user information such as profile picture, name, role, and social links in a clean and modern card layout.
## Features
* Clean and modern UI design
* Profile image display
* User name and designation
* Social media links
## Technologies Used
* HTML5 – Structure of the card
* CSS3 – Styling, layout
* Image – Profile picture
## Project Structure
**profile_card/** #project folder name
* faiz.jpg
* index.html
* style.css
## How to Run
1. Download or clone the project: git clone https://github.com/AL-FAIZKHAN007/act3_profile_card.git
2. Open the index.html file in any web browser
